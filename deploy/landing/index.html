Personal cloud backup
to S3 Glacier — cheap and permanent

A self-hosted backup system that syncs files from external hard drives to AWS S3 Glacier Instant Retrieval. Browse and stream your backed-up files from anywhere with a clean web UI. Install as a PWA on your phone.

Features

🚀

Fast Incremental Sync

SHA-256 computed in a single read pass during upload. Skips unchanged files. 10 parallel threads.

🔄

Move Detection

Reorganizing files locally triggers S3 CopyObject — no re-upload, instant and free.

📂

File Browser

Lazy-loaded folder tree, file search, image preview, video and audio playback, PDF and text viewer.

🎵

Media Player

Playlist mode, slideshow, shuffle, autoplay. WMA files transcoded on-the-fly via ffmpeg.

📱

PWA for Mobile

Installable on iOS and Android. Sidebar hidden on small screens. Touch-friendly resize handles.

💰

Ultra-low Cost

S3 Glacier Instant Retrieval at $0.004/GB/month. 350 GB costs ~$1.40/month. Millisecond retrieval.
